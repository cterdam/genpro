# 🔎 Lab

Lab (**L**anguage **A**I **B**uilding) allows you to systematically manage many
runs of training and evaluation of language models.

## 🏁 Setup

```zsh
conda create -n lab python=3.12
conda activate lab
pip install -r requirements.txt
```
